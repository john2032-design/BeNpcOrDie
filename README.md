# MooCore NPC or DIE

A comprehensive utility script for enhanced gameplay experience with advanced features.

## Features

- **ESP System**: Visual highlighting of opponents with team-based colors
- **Aim Assist**: Criminal targeting with advanced aim assistance
- **Auto Farm**: Automated money collection system
- **Auto Obby**: Automatic obstacle course completion
- **Player Utilities**: No Clip, God Mode, and character management
- **Task Automation**: Complete all tasks automatically
